# תהליך ייבוא קובצי PDF של Campus IL

המטרה היא להכניס לאפליקציה רק מידע שמקורו בקובצי PDF של `courses.campus.gov.il`, ולשמור לכל פריט מקור, עמוד וסטטוס בדיקה.

## קבצים שנוספו

- `data/sources/campus-il-pdfs.json` — מניפסט קובצי PDF רשמיים שנדונו ונמצאים בתהליך ייבוא.
- `data/extracted/pdfs/metadata.json` — מטא־דאטה מאומת על קובצי PDF, כולל מספר עמודים.
- `data/extracted/formulas/quant-summary-formulas.json` — נוסחאות ראשוניות המקושרות לדף הסיכום הכמותי ודורשות אימות עמוד מדויק.
- `data/extracted/pages/english-foundations-outline.json` — מבנה ראשוני של ספר יסודות אנגלית מתוך הטקסט שחולץ מה־PDF.
- `scripts/fetch-pdfs.js` — הורדת PDFים מ־Campus IL בלבד אל `.cache/pdfs`.
- `scripts/extract-pdf-text.js` — חילוץ טקסט מקומי אם מותקן `pdftotext`.

## מגבלת סביבת העבודה הנוכחית

בסביבה הנוכחית הורדת PDFים ישירות דרך הטרמינל נחסמה בגלל כשל DNS/Proxy. לכן נשמרו במאגר מניפסט, מטא־דאטה ומבנה נתונים ראשוני שאומתו דרך כלי הגישה לאינטרנט, והסקריפטים מוכנים להרצה בסביבת GitHub Actions או במכונה עם גישה תקינה לאינטרנט.

## מדיניות איכות

- אין לקבל URL שאינו מתחיל ב־`https://courses.campus.gov.il/`.
- אין לקבל פריט רשמי בלי `sourceId`.
- נוסחה או שאלה עם עמוד לא ידוע נשארת בסטטוס בדיקה ולא מוצגת כאילו אומתה סופית.
- תוכן AI נשאר במסלול נתונים נפרד ואינו נכנס למניפסט הרשמי.
